# Circle Area Calculator

A simple Python program that calculates the area of a circle using a user-provided radius. The program prompts the user to enter the radius in centimeters, applies the formula for the area of a circle using Python's built-in `math` module, rounds the result to two decimal places, and displays the calculated area in square centimeters.

## Features

* Accepts user input for the circle's radius.
* Uses the `math.pi` constant for accurate calculations.
* Calculates the area using the circle area formula.
* Rounds the result to two decimal places.
* Displays the area in square centimeters (cm²).

## How to Run

1. Make sure Python 3 is installed.
2. Run the script:

   ```bash
   python circle_area.py
   ```
3. Enter the radius in centimeters when prompted.

Welcome to counting area of circle
Input your radius in cm : 7
